# Smart Job Tracker API 🧠💼

A FastAPI-based backend that uses **Natural Language Processing (NLP)** to automatically analyze job descriptions and extract key information.

## 🚀 Features
- **AI Extraction:** Uses `DistilBERT-uncased` to identify Companies and Locations.
- **Hybrid Logic:** Custom keyword matching for Job Titles.
- **FastAPI Powered:** High-performance asynchronous API endpoints.
- **Developer Friendly:** Swagger UI documentation built-in.
